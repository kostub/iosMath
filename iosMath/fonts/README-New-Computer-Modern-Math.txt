New Computer Modern Math
========================

Author:  Antonis Tsolomitis
License: GUST Font License (GFL) — see GUST-FONT-LICENSE.txt

Version: 4.0

Upstream: https://ctan.org/pkg/newcomputermodern

New Computer Modern Math is the math companion font for the
New Computer Modern family. It provides a complete OpenType MATH
table and covers the full set of LaTeX mathematical symbols.
