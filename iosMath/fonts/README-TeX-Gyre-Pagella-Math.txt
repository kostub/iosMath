TeX Gyre Pagella Math
=====================

Version: 1.632

Authors: Bogusław Jackowski, Janusz M. Nowacki, Piotr Strzelczyk
         (GUST e-foundry)

Upstream: https://ctan.org/pkg/tex-gyre-math

License: GUST Font License (GFL)

TeX Gyre Pagella Math is part of the TeX Gyre Math collection, which
provides OpenType math fonts based on URW Palladio L (a Palatino
clone). It is designed to complement the TeX Gyre Pagella text font.

Downloaded from: https://www.gust.org.pl/projects/e-foundry/tg-math/download/texgyrepagella-math-1632.zip
